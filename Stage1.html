HTML - CSS - DOM

HTML: HyperText Markup Language
The standard markup language used to creat web page.
CSS: Cascading Style Sheets.
Style sheet language used for designing the look and formation of a document written in a markup language.
DOM: Document Object Model
A cross platform and language independent conversation for representing and interacting with objects in HTML (and other markup language). The nodes of every DOC are organised in a tree srtucture called DOM tree.
The browser turns the tag into elements that forms tree. the browser is able to make a tree is beacause of "DOM".

Avoiding Repetition

Avoiding repetition in our coding has lots and lots of benefits. As we'll learn ahead our coding will get more complex and bigger. Thus writing a structured code and avoiding repetition of same code will help us in:

  • Making less mistakes
  • Spotting the errors will be easy
  • Saves lots of time
Power Of CSS.

As we have learned above that CSS(Cascading Style Sheets) are used to decorate and make our web page look nice and beautiful and colorful. For example if i want my all the headings to be "Red" and "italic" and "underlined", then simply I'll use CSS for it and thus I'll not have to repeat the coding for all of my headings and instead type it once and then I'll connect my CSS with my HTML.
